h1. Nominatim Java API

This library provides a client for [http://wiki.openstreetmap.org/wiki/Nominatim Nominatim]. This API provides search and reverse geocoding operations. 

* nominatim-java-api is a pure Java implementation and relies on @org.json:json-20070829@ and @org.apache.httpcomponents:httpclient-4.0.2@
* nominatim-android-api uses the same source code but depends on @com.google.android:android-2.3.3@

Please report bugs on [https://github.com/kops/nominatim-java-api/issues/new Github tracker].
